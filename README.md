# ESP32-SIC
Repositorio dedicado a organizar y documentar los programas creados/modificados a utilizar con la ESP32-CAM para el diplomado Samsung Innovation Campus

## Equipo utilizado
- Protoboard
- Jumpers macho-macho
- Jumpers macho-hembra
- Jumpers hembra-hembra
- LEDs de colores
- ESP32-CAM
- FTDI FT232rl
- Conocimientos
- Actitud
